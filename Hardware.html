What skill set do you believe or know is required for this area of study? In this field, you require both technical and soft skills. These range from cloud computing, data analysis, coding and cybersecurity all the way to communication, organization, adaptability, and problem solving.

What is the job description of the employee/manager in this area of study? Firstly, the IT Manager oversees the entire organization's technology as well as all IT related staff. Additionally, they are responsible for maintaining and implementing networks, software, and hardware to meet the business goals.

One of the most essential aspects of IT is the hardware that allows devices to simply function. Hardware refers to the physical components of any electronic device. Anything you can touch regarding technology is considered hardware.

Hardware in Information Technology can be categorized into:
    Input Devices
    Output Devices
    Processing Devices
    Storage Devices
    Networking Devices
    Peripheral Devices

a) Input Devices These devices are used to enter data into the computer system.

    Keyboard
    Mouse
    Scanner
    Webcam
    Microphone
    Touchscreen

b) Output Devices These devices display the results of processed data.

    Monitor
    Printer
    Speakers
    Projector

c) Processing Unit This is the brain of the computer, responsible for executing instructions.

    Central Processing Unit (CPU)
    Graphics Processing Unit (GPU)
    Motherboard
    RAM (Random Access Memory
)

d) Storage Devices Used to store data and information.

    Hard Disk Drive (HDD)
    Solid State Drive (SSD)
    Optical Discs (CD/DVD)
    USB Flash Drives
    Memory Cards

e) Networking Hardware Helps in connecting computers and sharing resources.

    Router
    Modem
    Network Interface Card (NIC)
    Switch
    Hub

f) Peripheral Devices Additional devices that expand functionality.

    External Hard Drives
    Game Controllers
    External DVD Drives
